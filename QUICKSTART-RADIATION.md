# 辐射暴露管理快速指南

本文档介绍如何使用医学辐射暴露管理功能。

## 前置条件

在使用辐射管理功能前，必须先设置用户基础参数：

```bash
/profile set 175 70 1990-01-01
```

参数说明：
- `175`: 身高（厘米）
- `70`: 体重（公斤）
- `1990-01-01`: 出生日期

## 基础使用

### 1. 添加辐射检查记录

```bash
# 记录今天的检查（默认日期为今天）
/radiation add CT 胸部
/radiation add X光 胸部

# 记录指定日期的检查
/radiation add CT 腹部 2025-12-30
/radiation add PET-CT 全身 2025-12-25
```

### 2. 查看累积状态

```bash
/radiation status
```

输出示例：

## 辐射剂量调整机制

系统会根据您的体表面积自动调整辐射剂量：

1. **计算体表面积**（使用 Mosteller 公式）：
   ```
   BSA = √(身高cm × 体重kg / 3600)
   ```

2. **调整系数**：
   ```
   系数 = 实际BSA / 标准BSA(1.73 m²)
   ```

3. **实际剂量**：
   ```
   实际剂量 = 标准剂量 × 系数
   ```

示例：
- 标准体型（BSA 1.73 m²）：胸部CT = 7.0 mSv
- 较大体型（BSA 2.0 m²）：胸部CT = 8.1 mSv
- 较小体型（BSA 1.5 m²）：胸部CT = 6.1 mSv

## 辐射衰减规律

系统采用指数衰减模型计算往年辐射残留：

```
残留剂量 = 初始剂量 × (0.5)^(经过年数)
```

衰减示例：
- 1年前的检查：残留 50%
- 2年前的检查：残留 25%
- 3年前的检查：残留 12.5%
- 4年前的检查：残留 6.25%

## 安全阈值参考

| 累积剂量 | 风险级别 | 建议 |
|---------|---------|------|
| < 1 mSv/年 | ✅ 安全 | 无需担心 |
| 1-10 mSv/年 | ⚠️ 关注 | 记录并定期检查 |
| 10-50 mSv/年 | ⚠️ 警告 | 咨询医生，考虑替代方案 |
| > 50 mSv/年 | 🚨 危险 | 立即咨询医生 |

## 常见问题

### Q：为什么要记录辐射暴露？
A：医学影像检查（特别是CT）会产生电离辐射，累积辐射剂量可能增加健康风险。记录和追踪有助于：
- 了解自己的累积剂量
- 与医生讨论检查的必要性
- 考虑低剂量或无辐射的替代检查

### Q：辐射剂量是如何计算的？
A：系统基于医学标准的辐射剂量参考值，并根据您的体表面积自动调整。

### Q：什么是辐射衰减？
A：人体会逐渐代谢辐射影响。本系统采用50%/年的衰减率估算残留剂量。

### Q：多大的辐射剂量是安全的？
A：
- 自然本底辐射：约 2.4 mSv/年
- 公众年剂量限值：1 mSv/年（除自然本底外）
- 一般建议：年累积 < 10 mSv 为相对安全范围

### Q：可以更新基础参数吗？
A：可以。使用 `/profile set` 命令更新身高体重，系统会重新计算体表面积和辐射剂量。

### Q：如何清空所有辐射记录？
A：使用 `/radiation clear` 命令（谨慎使用）。

## 典型使用场景

### 场景1：常规体检记录

```bash
# 体检后记录各项检查
/profile view  # 确认基础参数已设置
/radiation add CT 胸部
/radiation add X光 胸部
/radiation status  # 查看累积情况
```

### 场景2：定期随访

```bash
# 每次检查后记录
/radiation add CT 腹部 2025-12-01

/radiation add CT 腹部 2025-12-30

# 年底查看年度总结
/radiation status
/radiation history
```

### 场景3：评估检查必要性

```bash
# 医生建议做CT前，查看累积情况
/radiation status

# 如果累积量高，与医生讨论：
# 1. 是否可以用MRI（无辐射）
# 2. 是否可以用低剂量CT
# 3. 检查是否可以延后
```

## 注意事项

1. **首次使用必读**：必须先设置基础参数（身高、体重、出生日期）
2. **仅供参考**：本系统仅供参考，不作为医疗诊断依据
3. **咨询医生**：所有诊疗决策请咨询专业医生
4. **定期备份**：建议定期备份 `data/` 目录
5. **准确记录**：尽量准确记录检查类型和部位
6. **及时更新**：体重变化较大时，更新基础参数
