## 场景1：高血压患者的日常管理

### 背景信息

**患者：张先生，45岁**

- 诊断：原发性高血压（2年）
- 当前用药：氨氯地平 5mg 每日1次
- 血压目标：<130/80 mmHg（一般人群标准）
- 风险因素：吸烟（20支/天，25包年）
- 合并症：无

### 第一阶段：初始评估

#### 步骤1：查看当前高血压质量状态

```
/quality hypertension status
```

**输出解读：**
- 平均血压：142/88 mmHg（2级高血压）
- 达标率：62%（较差）
- 质量等级：D
- 家庭监测依从性：85%（良好）

**关键发现：**
- ✅ 家庭监测习惯良好
- ❌ 血压控制未达标
- ❌ 收缩压和舒张压均偏高

#### 步骤2：查看血压趋势

```
/quality hypertension trends 180
```

**输出解读：**
- 6个月趋势：138→142 mmHg（收缩压上升）
- 趋势：恶化中 📈
- 可能原因：药物效果不足、生活方式因素

#### 步骤3：获取个性化建议

```
/quality recommendations hypertension
```

**优先建议：**
1. 🚨 **立即戒烟**（最大获益潜力）
2. ⚠️ **调整降压药物**（与医生讨论）
3. ⚠️ **开始DASH饮食**（可降低6-11 mmHg）
4. ⚠️ **规律运动**（每周150分钟）

### 第二阶段：干预实施

#### 行动1：开始DASH饮食

**DASH饮食核心原则：**
- ✅ 高蔬菜水果（每天4-5份）
- ✅ 低脂乳制品（每天2-3份）
- ✅ 全谷物（每天6-8份）
- ❌ 限制钠盐（每天<1500mg ≈ 2/3茶匙）
- ❌ 限制加工食品

**记录饮食改变：**
```
/diet lunch 燕麦粥、低脂牛奶、苹果
/diet dinner 烤鸡胸、糙米饭、西兰花、低脂酸奶
```

#### 行动2：开始规律运动

**运动计划：**
- 周一：快走30分钟
- 周三：游泳30分钟
- 周五：快走30分钟
- 周六：骑行45分钟

**记录运动：**
```
/symptom 今天快走30分钟，心率110，感觉良好
```

#### 行动3：咨询医生调整药物

**准备家庭血压记录：**
```
/vitals history bp 30 > bp-report.txt
```

**就诊前准备：**
- 打印最近30天血压记录
- 准备询问的问题：
  - 是否需要增加氨氯地平剂量至10mg？
  - 是否需要联合第二种降压药？
  - 当前血压是否增加心血管风险？

**就诊后更新：**
（医生决定：氨氯地平增加至10mg，每日1次）

```
/medication add 氨氯地平 10mg 每天1次 早餐后服用
```

#### 行动4：开始戒烟计划

```
/symptom 开始戒烟，设定戒烟日期：2025-02-01
/symptom 戒烟第1天，使用尼古丁贴片
```

### 第三阶段：持续监测

#### 每日血压监测

**早晨测量（起床后、早餐前、服药前）：**
```
/vitals bp 128/82 早上
```

**晚上测量（睡前、服药后）：**
```
/vitals bp 132/85 晚上
```

#### 每周质量评估

```
/quality hypertension status
```

**第4周评估：**
- 平均血压：136/85 mmHg
- 达标率：68%
- 趋势：改善中 📉
- 变化：-6/-3 mmHg

**第8周评估：**
- 平均血压：132/84 mmHg
- 达标率：75%
- 趋势：持续改善 📉
- 变化：-10/-4 mmHg

**第12周评估：**
- 平均血压：128/81 mmHg
- 达标率：85%
- 趋势：显著改善 📉
- 质量等级：B ✅
- **接近目标值！**

### 第四阶段：维持阶段

#### 维持健康习惯

**继续坚持：**
- ✅ 家庭血压监测（早晚各一次）
- ✅ DASH饮食模式
- ✅ 规律运动（每周150分钟）
- ✅ 戒烟（已戒3个月）
- ✅ 规律服用降压药

#### 定期并发症筛查

```
/quality complications hypertension
```

**年度检查计划：**
- ✅ 心血管评估（2025-01-15完成）
- ✅ 肾功能检查（2025-02-20完成）
- ✅ 眼底检查（2025-06-10完成）

#### 长期质量跟踪

```
/quality hypertension trends 365
```

**年度总结：**
- 年初平均血压：142/88 mmHg
- 年末平均血压：126/80 mmHg
- 改善：-16/-8 mmHg
- 质量等级：D → A ✅
- **显著降低心血管风险！**

## 场景2：糖尿病患者的质量控制

### 背景信息

**患者：李女士，58岁**

- 诊断：2型糖尿病（5年）
- 当前用药：二甲双胍 500mg 每日2次
- HbA1c目标：<7.0%（一般成人标准）
- 合并症：高血压（已控制）
- 并发症：无

### 第一阶段：糖代谢控制评估

#### 步骤1：查看糖尿病质量状态

```
/quality diabetes status
```

**输出解读：**
- 最近HbA1c：7.5%（未达标）
- 平均空腹血糖：7.8 mmol/L（偏高）
- 血糖达标率：68%（需改进）
- 质量等级：C

**关键发现：**
- ❌ HbA1c高于目标值
- ❌ 空腹血糖控制不佳
- ✅ 无严重低血糖事件
- ⚠️ 餐后血糖控制不稳定

#### 步骤2：查看血糖趋势

```
/quality diabetes trends
```

**输出解读：**
- 过去6个月HbA1c：7.2% → 7.3% → 7.5%
- 趋势：缓慢上升 📈
- 可能原因：
  - 饮食控制不严格
  - 药物效果不足
  - 胰岛功能逐渐衰退

#### 步骤3：分析血糖模式

**查看最近1周血糖记录：**
```
/vitals history glucose 7
```

**发现的模式：**
- 空腹血糖：7-8 mmol/L（偏高）
- 早餐后血糖：9-11 mmol/L（波动大）
- 午餐后血糖：8-10 mmol/L（稍高）
- 晚餐后血糖：8-9 mmol/L（接近目标）
- 睡前血糖：6-7 mmol/L（良好）

**问题分析：**
1. 空腹血糖偏高 → 肝糖输出增加
2. 早餐后峰值高 → 早餐碳水化合物过多
3. 午餐后控制改善 → 午餐结构合理
4. 晚餐后控制最好 → 晚餐最清淡

### 第二阶段：干预调整

#### 行动1：调整饮食结构

**问题：早餐碳水化合物过多**
- 当前早餐：白粥、馒头、咸菜
- 调整后：燕麦粥、煮鸡蛋、牛奶、坚果

**记录改进：**
```
/diet breakfast 燕麦粥50g、煮鸡蛋1个、低脂牛奶250ml、核桃5个
```

**问题：全天热量分配不合理**
- 当前：早餐30%、午餐30%、晚餐40%
- 调整：早餐25%、午餐40%、晚餐30%、加餐5%

**记录加餐：**
```
/diet snack 上午10点苹果1个
/diet snack 下午3点无糖酸奶100g
```

#### 行动2：调整运动时机

**增加餐后运动：**
- 早餐后：快走20分钟
- 晚餐后：散步30分钟

**记录：**
```
/symptom 早餐后快走20分钟，心率105，感觉良好
```

#### 行动3：咨询医生调整药物

**当前用药：二甲双胍 500mg 每日2次**

**与医生讨论的选项：**
1. 增加二甲双胍剂量至1000mg 每日2次
2. 或改用二甲双胍缓释片
3. 或加用DPP-4抑制剂（如西格列汀）

**就诊后更新：**
（医生决定：二甲双胍增加至1000mg 每日2次）

```
/medication update 二甲双胍 1000mg 每天2次 餐后服用
```

### 第三阶段：血糖精细化管理

#### 每日血糖监测计划

**7点血糖谱（每周1-2天）：**
```
/vitals glucose 6.8 2025-01-10 07:00 空腹
/vitals glucose 9.5 2025-01-10 08:30 早餐后2小时
/vitals glucose 7.2 2025-01-10 11:30 午餐前
/vitals glucose 8.8 2025-01-10 14:30 午餐后2小时
/vitals glucose 6.5 2025-01-10 17:30 晚餐前
/vitals glucose 8.2 2025-01-10 20:30 晚餐后2小时
/vitals glucose 5.8 2025-01-10 22:30 睡前
```

#### 每周质量评估

```
/quality diabetes status
```

**第4周评估：**
- 平均空腹血糖：7.2 mmol/L（改善）
- HbA1c预测：7.3%（改善）
- 血糖达标率：75%（提升）
- 趋势：改善中 ✅

**第8周评估：**
- 平均空腹血糖：6.8 mmol/L（接近目标）
- 实际HbA1c：7.0%（达标！）✅
- 血糖达标率：82%（良好）
- 质量等级：B

**第12周评估：**
- 平均空腹血糖：6.5 mmol/L（目标范围内）
- HbA1c：6.8%（优秀）✅
- 血糖达标率：88%（优秀）
- 质量等级：A ✅
- **达到糖代谢控制目标！**

### 第四阶段：并发症预防

#### 年度并发症筛查


**筛查项目完成情况：**

| 项目 | 检查日期 | 结果 | 下次检查 |

**并发症风险评估：**

## 场景3：高血压+糖尿病合并管理

### 背景信息

**患者：王先生，62岁**

- 诊断：2型糖尿病（10年）、高血压（8年）
- 当前用药：
  - 二甲双胍 1000mg 每日2次
  - 氨氯地平 5mg 每日1次
- 血压目标：<130/80 mmHg（糖尿病目标）
- HbA1c目标：<7.0%

### 综合质量评估

#### 步骤1：查看整体状态

```
/quality status
```

**高血压管理：**
- 平均血压：138/82 mmHg
- 达标率：70%
- 问题：收缩压未达标

**糖尿病管理：**
- HbA1c：7.8%
- 血糖达标率：65%
- 问题：HbA1c偏高，血糖波动大

**关键问题：**
1. 血压和血糖均未达标
2. 需要优先控制哪个？
3. 两病相互影响，需综合管理

#### 步骤2：获取综合建议

```
/quality recommendations
```

**优先级排序：**
1. 🚨 **严格控制血压**（糖尿病患者血压目标更低）
2. ⚠️ **降低HbA1c至7.0%以下**
3. ⚠️ **控制血脂**（糖尿病患者需更严格控制）
4. ⚠️ **体重管理**（如超重或肥胖）

**综合干预策略：**
- 血压优先：收缩压每降低5 mmHg，心血管事件降10%
- 血糖目标：HbA1c降至7.0%以下
- 血脂目标：LDL降至<1.8 mmol/L（糖尿病患者更严格）

### 综合干预计划

#### 第1-3个月：血压达标优先

**药物调整：**
```
/medication update 氨氯地平 10mg 每天1次
/medication add 厄贝沙坦 150mg 每天1次
```

**生活方式：**
- DASH饮食（限钠<1500mg/天）
- 规律运动（每周150分钟中等强度）
- 戒烟限酒
- 减重5-10%

**预期效果：**
- 血压降至130/80以下
- 心血管风险降低20-30%

#### 第4-6个月：血糖达标

**药物调整：**
```
/medication add 西格列汀 100mg 每天1次
```

**饮食调整：**
- 糖尿病饮食 + DASH饮食（低钠、低脂、控糖）
- 碳水化合物：占总热量45-50%
- 蛋白质：占总热量15-20%
- 脂肪：占总热量30%（以不饱和脂肪为主）

**预期效果：**
- HbA1c降至7.0%以下
- 血糖波动减少
- 低血糖风险低

#### 第7-12个月：长期维持

**综合达标目标：**
- ✅ 血压：<130/80 mmHg
- ✅ HbA1c：<7.0%
- ✅ LDL：<1.8 mmol/L
- ✅ 体重：BMI 18.5-24.9
- ✅ 并发症筛查：全部按时完成

## 场景5：生活方式干预跟踪

### 背景：通过生活方式改善血压

**患者：陈女士，48岁**

- 诊断：高血压（1级）
- 当前血压：138/86 mmHg
- 危险因素：超重（BMI 28）、久坐、饮食高盐
- 目标：通过生活方式改善血压，避免药物治疗

### 基线评估

#### 第1周：基线数据收集

**家庭血压监测（早晚各1次，连续7天）：**
```
/vitals bp 140/88 早上
/vitals bp 136/86 晚上
```

**7天平均：**
- 早晨平均：139/87 mmHg
- 晚间平均：135/85 mmHg
- 7天总平均：137/86 mmHg

**基线达标率：** 15%（非常低）

### 生活方式干预计划

#### 干预1：DASH饮食

**钠盐摄入目标：** <1500mg/天

**具体措施：**
- ❌ 不吃加工食品（香肠、火腿、罐头）
- ❌ 不吃腌制食品（咸菜、酱菜）
- ❌ 烹饪时少放盐（每天<5g）
- ✅ 多用香料、柠檬、醋调味

**记录饮食：**
```
/diet lunch 烤鸡胸、糙米饭、清炒西兰花（无盐）
/diet dinner 蒸鱼、糙米饭、凉拌黄瓜（醋调味）
```

#### 干预2：减重

**目标：** 减重5-10%（7-14kg）
**计划：** 每月减2-3kg，持续3-6个月

### 使用流程总结

1. **初始评估：** `/quality [disease] status`

### 关键成功因素

✅ **坚持家庭监测**（依从性>90%）
✅ **记录完整准确**（不遗漏测量）
✅ **定期评估质量**（每1-3个月）
✅ **遵循循证建议**（基于指南）
✅ **与医生沟通**（记录带去就诊）
✅ **生活方式干预**（饮食、运动、减重）

### 常见问题

**Q：血压/血糖波动大怎么办？**
A：检查测量方法、饮食、运动、药物依从性，记录7天详细数据后咨询医生。

**Q：多久能达标？**
A：一般需要3-6个月。生活方式干预见效慢但效果持久，药物调整见效快但需个体化。

**Q：如何判断需要就医？**
A：如出现持续升高、严重症状、并发症迹象，立即就医。

**Q：可以自行调整药物吗？**
A：不可以。所有药物调整必须在医生指导下进行。
