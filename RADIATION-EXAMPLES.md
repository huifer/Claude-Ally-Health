## 示例5：不同体型用户的剂量对比

### 体型较小用户（160cm, 50kg）

```bash
/profile set 160 50 1995-01-01
/radiation add CT 胸部
```

**计算：**
- 体表面积：1.51 m²
- 调整系数：1.51 / 1.73 = 0.87
- 实际剂量：7.0 × 0.87 = 6.1 mSv

### 体型标准用户（175cm, 70kg）

```bash
/profile set 175 70 1990-01-01
/radiation add CT 胸部
```

**计算：**
- 体表面积：1.85 m²
- 调整系数：1.85 / 1.73 = 1.07
- 实际剂量：7.0 × 1.07 = 7.5 mSv

### 体型较大用户（185cm, 95kg）

```bash
/profile set 185 95 1985-01-01
/radiation add CT 胸部
```

**计算：**
- 体表面积：2.21 m²
- 调整系数：2.21 / 1.73 = 1.28
- 实际剂量：7.0 × 1.28 = 9.0 mSv

## 示例6：实际使用场景

### 场景：随访肺部结节

患者每3个月做一次胸部CT随访：

```bash
# 第1次检查
/radiation add CT 胸部 2025-01-15

# 第2次检查（3个月后）
/radiation add CT 胸部 2025-04-20

# 第3次检查（6个月后）
/radiation add CT 胸部 2025-07-25

# 第4次检查（9个月后）
/radiation add CT 胸部 2025-10-30

# 查看年度累积
/radiation status
```

**结果：** 本年累积约 30 mSv，超过建议安全范围

**医生讨论：**
1. 是否可以改用低剂量CT（剂量减半）
2. 是否可以延长随访间隔
3. 是否可以用其他检查替代

## 示例8：错误处理

### 未设置基础参数

```bash
# 清空profile后尝试添加辐射记录
/radiation add CT 胸部
```

**错误输出：**
```
❌ 错误：未找到用户基础参数

请先设置基础参数：
/profile set 175 70 1990-01-01

身高体重用于计算体表面积，以准确评估辐射剂量
```

### 日期格式错误

```bash
/radiation add CT 胸部 2025/12/31
```

**错误输出：**
```
❌ 错误：日期格式不正确

正确格式：YYYY-MM-DD（例如：2025-12-31）
```

### 不支持的检查类型

```bash
/radiation add MRI 脑部
```

**错误输出：**
```
⚠️ 警告：MRI检查无电离辐射

MRI（磁共振成像）不产生电离辐射，无需记录到辐射暴露管理系统。

支持的检查类型：CT、X光、PET-CT、骨扫描、血管造影等
```

## 命令速查表

| 命令 | 功能 | 示例 |
|------|------|------|
| `/profile set` | 设置基础参数 | `/profile set 175 70 1990-01-01` |
| `/profile view` | 查看基础参数 | `/profile view` |
| `/radiation add` | 添加辐射记录 | `/radiation add CT 胸部` |
| `/radiation status` | 查看累积状态 | `/radiation status` |
| `/radiation history` | 查看历史记录 | `/radiation history` |
| `/radiation clear` | 清空记录 | `/radiation clear` |

## 最佳实践

1. **首次使用**：先设置准确的基础参数
2. **及时记录**：每次检查后立即记录
3. **定期查看**：每季度查看一次累积状态
4. **医生沟通**：高剂量检查前先查看累积情况
5. **数据备份**：定期备份 `data/` 目录
