# 疾病风险评估

基于多维度健康数据进行疾病风险评估和预测，帮助您了解当前的健康风险，并指导预防干预策略。

## 使用方法

### 基本用法

```
/think-risk cardiovascular
/think-risk comprehensive
/think-risk stroke
```

### 心血管风险评估（最常用）

```
/think-risk cardiovascular
```

系统会自动读取您的健康数据并评估ASCVD（动脉粥样硬化性心血管疾病）10年风险。

**需要的参数：**
- 年龄（40-79岁）
- 性别
- 收缩压
- 总胆固醇
- HDL-C
- 是否吸烟
- 是否糖尿病
- 是否服用降压药

### 综合风险评估

```
/think-risk comprehensive
```

评估多种疾病风险，包括心血管、脑卒中、糖尿病等。

## 执行流程

疾病风险评估会执行以下步骤：

### 1. 数据整合

系统会从多个数据源整合您的健康信息：

- **基础档案**：年龄、性别、家族史
- **慢性病信息**：高血压、糖尿病诊断
- **生命体征历史**：血压、血糖趋势
- **生活习惯**：吸烟、运动、饮食
- **检查结果**：血脂、血糖等

### 2. 风险因素识别

系统会识别：

#### 不可改变因素
- 年龄
- 性别
- 家族史

#### 可改变因素
- 高血压
- 血脂异常
- 吸烟
- 糖尿病
- 肥胖
- 缺乏运动

### 3. ASCVD风险计算

使用**2013 ACC/AHA指南**的Pooled Cohort Equations计算10年ASCVD风险。

**适用范围**：
- 年龄：40-79岁
- 种族：白种人/黑人（其他种族使用近似值）
- 评估：心梗、冠心病死亡、缺血性脑卒中

**计算公式**：
```
10年风险 = 1 - 0.9573^exp(ΣβiXi - 28.1567)
```

其中β是回归系数，X是各危险因素的值。

### 4. 风险分层

根据10年风险百分比进行分层：

| 风险分层 | 10年风险 | 颜色 | 建议 |
|---------|---------|------|------|
| 低风险 | <5% | 🟢 绿色 | 保持健康生活方式 |
| 边界风险 | 5-7.5% | 🟡 黄色 | 建议生活方式干预 |
| 中等风险 | 7.5-10% | 🟠 橙色 | 考虑药物治疗 |
| 高风险 | 10-20% | 🔴 红色 | 建议药物治疗 |
| 很高风险 | >20% | 🟤 深红 | 积极药物治疗，强化控制 |

### 5. 干预效果预测

系统会预测不同干预措施对风险的影响：

**戒烟效果：**
- 可降低约50%心血管风险
- 戒烟5年内风险显著降低
- **最高优先级干预**

**血压控制效果：**
- 血压降至<130/80 mmHg
- 可降低约20-30%心血管风险
- ACEI/ARB类药物首选

**他汀类药物效果：**
- 中等强度他汀（如阿托伐他汀10-20mg）
- 可降低约25-30%心血管风险
- 1-2年内见效

**生活方式综合干预：**
- DASH饮食 + 运动减重 + 戒烟限酒
- 可降低约30%风险
- 3-6个月见效

### 6. 个性化建议生成

基于您的风险分层和危险因素，系统会生成：

#### 优先级1 - 立即行动

- 戒烟（如适用）
- 血压控制（如血压升高）
- 血脂管理（如LDL-C升高）

#### 优先级2 - 近期改进

- DASH饮食
- 规律运动
- 体重管理
- 限制饮酒

#### 优先级3 - 长期维持

- 定期监测
- 药物依从性
- 压力管理

## 注意事项

### ⚠️ 重要提示

1. **模型限制**：ASCVD风险计算适用于40-79岁白种人，其他人种使用近似值
2. **仅供参考**：风险评估仅供参考，不能替代专业医疗建议
3. **个体差异**：每个人的风险因素影响程度不同
4. **动态变化**：风险会随时间变化，建议定期重新评估

### ✅ 适用场景

- 40-79岁成人心血管风险评估
- 健康体检后的风险评估
- 生活方式干预前的基线评估
- 药物治疗前的风险-获益分析

### ❌ 不适用场景

- 已确诊心血管疾病（需二级预防模型）
- 年龄<40岁或>79岁（使用其他模型）
- 已服用他汀类药物（需重新评估）

## 技术实现

疾病风险评估基于以下技术：

1. **2013 ACC/AHA Pooled Cohort Equations**：经过验证的ASCVD风险计算模型
2. **风险分层算法**：基于指南的风险分层标准
3. **干预效果预测模型**：基于循证医学的干预效果估计
4. **多维度数据聚合**：整合健康档案、生命体征、检查结果等
